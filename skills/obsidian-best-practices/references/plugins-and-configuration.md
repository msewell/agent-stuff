## Core Plugins to Enable

Enable these in Settings → Core plugins:

| Plugin | Why |
|---|---|
| **Daily Notes** | Creates a new note for each day. Starting point for most workflows. |
| **Templates** | Insert predefined note structures. Use alongside Templater for more power. |
| **Backlinks** | Shows every note that links to the current note. Essential for a linked knowledge base. |
| **Outgoing Links** | Shows what the current note links to. Useful for reviewing context. |
| **Tags** | Browse and filter by tags. |
| **Graph View** | Visualize your vault as a node graph. Spot clusters and orphan notes. |
| **Command Palette** | `Ctrl/Cmd + P` — keyboard-driven launcher for everything. |
| **Page Preview** | Hover over a link to preview the note without navigating away. |
| **Bookmarks** | Pin frequently used notes, searches, or graph views. |
| **Canvas** | Infinite spatial canvas for visual thinking and mapping. |
| **Properties View** | Browse and manage YAML frontmatter across your vault. |

## Essential Community Plugins

Install via Settings → Community plugins → Browse.

### Tier 1: Install These First

| Plugin | Purpose |
|---|---|
| **[Dataview](https://github.com/blacksmithgu/obsidian-dataview)** | Query your vault like a database. Tables, lists, and task views from frontmatter and inline fields. |
| **[Templater](https://github.com/SilentVoid13/Templater)** | Advanced templates with JavaScript execution, dynamic dates, file creation, and conditional logic. Far more powerful than core Templates. |
| **[Obsidian Git](https://github.com/Vinzent03/obsidian-git)** | Automatic Git commit, push, pull, and diff — all within Obsidian. |
| **[Tasks](https://github.com/obsidian-tasks-group/obsidian-tasks)** | Rich task management with due dates, recurrence, priorities, and global queries. |
| **[Calendar](https://github.com/liamcain/obsidian-calendar-plugin)** | Calendar sidebar for navigating and creating daily/weekly notes. |

### Tier 2: High-Value Additions

| Plugin | Purpose |
|---|---|
| **[Excalidraw](https://github.com/zsviczian/obsidian-excalidraw-plugin)** | Embedded whiteboard drawings — architecture diagrams, flowcharts, system designs. |
| **[Kanban](https://github.com/mgmeyers/obsidian-kanban)** | Markdown-backed Kanban boards for visual task tracking. |
| **[Periodic Notes](https://github.com/liamcain/obsidian-periodic-notes)** | Extends Daily Notes with weekly, monthly, quarterly, and yearly note templates. |
| **[Advanced Tables](https://github.com/tgrosinger/advanced-tables-obsidian)** | Tab-to-navigate Markdown tables, auto-formatting, sorting, and formulas. |
| **[Linter](https://github.com/platers/obsidian-linter)** | Auto-format Markdown on save — consistent headings, spacing, frontmatter ordering. |

### Tier 3: When You Need Them

| Plugin | Purpose |
|---|---|
| **[DB Folder](https://github.com/RafaelGB/obsidian-db-folder)** | Notion-like database views from a folder of notes. |
| **[Mermaid Tools](https://github.com/dartungar/obsidian-mermaid)** | Enhanced Mermaid diagram editing (Obsidian has native Mermaid rendering). |
| **[Quick Add](https://github.com/chhoumann/quickadd)** | Macros, capture shortcuts, and multi-step workflows from the command palette. |
| **[Various Complements](https://github.com/tadashi-aikawa/obsidian-various-complements-plugin)** | Auto-complete for words, links, and frontmatter values as you type. |

## Plugin Discipline

Don't hoard plugins. Install what you need, not what looks cool. Each plugin adds startup time and maintenance burden. Audit installed plugins quarterly — disable any you haven't used in 3 months.

## Keyboard Shortcuts Worth Setting

| Action | Suggested Shortcut |
|---|---|
| Open daily note | `Ctrl/Cmd + Shift + D` |
| Quick switcher (search notes) | `Ctrl/Cmd + O` |
| Insert template | `Alt + T` |
| Toggle edit/preview | `Ctrl/Cmd + E` |
| Open command palette | `Ctrl/Cmd + P` |
| Create new note | `Ctrl/Cmd + N` |
| Open graph view | `Ctrl/Cmd + G` |
| Navigate back | `Ctrl/Cmd + Alt + ←` |
